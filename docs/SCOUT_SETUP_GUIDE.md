# Настройка Агента-Скаута в n8n

**Дата:** 2026-01-19
**Workflow:** scout-agent-v1.json
**Статус:** Ready to deploy

---

## 📋 Что делает этот workflow?

Агент-Скаут автоматически:

1. **Сканирует AliExpress** через SerpAPI (100 товаров за раз)
2. **Фильтрует по критериям:** рейтинг ≥4.5, отзывы ≥100, цена $15-60
3. **Проверяет тренды** через Google Trends
4. **Анализирует через GPT-4** креативный потенциал
5. **Рассчитывает юнит-экономику** (прибыль, маржа, ROI)
6. **Сохраняет результаты** в Google Sheets
7. **Отправляет уведомления** в Telegram о находках STRONG BUY

## 🛠️ Что нужно для запуска

### 1. API ключи

#### SerpAPI (ОБЯЗАТЕЛЬНО)
- **Зачем:** Парсинг AliExpress, Amazon
- **Цена:** $50/месяц (100 запросов/день)
- **Регистрация:** https://serpapi.com/
- **Как получить:**
  1. Зарегистрироваться
  2. Dashboard → API Key
  3. Скопировать ключ

#### OpenAI API (ОБЯЗАТЕЛЬНО)
- **Зачем:** GPT-4 анализ креативного потенциала
- **Цена:** ~$5-10/месяц (зависит от использования)
- **Регистрация:** https://platform.openai.com/
- **Как получить:**
  1. Создать аккаунт
  2. API Keys → Create new secret key
  3. Скопировать ключ

#### Google Sheets API (ОБЯЗАТЕЛЬНО)
- **Зачем:** Сохранение результатов
- **Цена:** БЕСПЛАТНО
- **Настройка:**
  1. Создать Google таблицу
  2. В n8n добавить Google Sheets credentials
  3. Дать доступ к таблице

#### Telegram Bot (ОПЦИОНАЛЬНО)
- **Зачем:** Уведомления о находках
- **Цена:** БЕСПЛАТНО
- **Создание бота:**
  1. Написать @BotFather в Telegram
  2. /newbot
  3. Получить токен
  4. Узнать свой Chat ID: @userinfobot

## 📥 Импорт workflow в n8n

### Шаг 1: Откройте n8n

Перейдите по адресу: http://localhost:5678

### Шаг 2: Импортируйте workflow

1. Нажмите **"Import from File"** или **Ctrl+O**
2. Выберите файл: `TARS/workflows/scout-agent-v1.json`
3. Workflow появится в редакторе

### Шаг 3: Настройте credentials

#### SerpAPI

1. Кликните на ноду **"SerpAPI: Поиск на AliExpress"**
2. В поле **Authentication** → **Create New Credential**
3. Выберите **HTTP Query Auth**
4. Добавьте параметр:
   - Name: `api_key`
   - Value: `ВАШ_SERPAPI_КЛЮЧ`

#### OpenAI

1. Кликните на ноду **"GPT-4: Слой 7"**
2. **Credentials** → **Create New**
3. Выберите **OpenAI**
4. API Key: `ВАШ_OPENAI_КЛЮЧ`

#### Google Sheets

1. Кликните на ноду **"Сохранить в Google Sheets"**
2. **Credentials** → **Create New**
3. Выберите **Google Sheets OAuth2**
4. Следуйте инструкциям OAuth
5. **Document ID:** Скопируйте ID вашей Google таблицы из URL
   - URL: `https://docs.google.com/spreadsheets/d/ВАША_ТАБЛИЦА_ID/edit`
   - Скопируйте `ВАША_ТАБЛИЦА_ID`

#### Telegram (опционально)

1. Кликните на ноду **"Уведомление в Telegram"**
2. **Credentials** → **Create New**
3. **Access Token:** Токен от @BotFather
4. **Chat ID:** Ваш Chat ID от @userinfobot

## ⚙️ Настройка параметров

### Изменить категорию поиска

В ноде **"Установить параметры поиска"**:

```javascript
category: "Home & Garden"  // Измените на нужную категорию
```

**Популярные категории:**
- `"Health & Beauty"`
- `"Sports & Entertainment"`
- `"Consumer Electronics"`
- `"Home Improvement"`
- `"Toys & Hobbies"`

### Изменить критерии фильтрации

```javascript
min_rating: 4.5,      // Минимальный рейтинг
min_reviews: 100,     // Минимум отзывов
min_price: 15,        // Минимальная цена
max_price: 60         // Максимальная цена
```

### Расписание запуска

По умолчанию: **каждый день в 9:00**

Чтобы изменить:
1. Кликните на **"Запуск каждый день в 9:00"**
2. Измените Cron Expression:
   - `0 9 * * *` - каждый день в 9:00
   - `0 9,15 * * *` - каждый день в 9:00 и 15:00
   - `0 9 * * 1-5` - по будням в 9:00

## 🔧 Troubleshooting

### Ошибка: "SerpAPI quota exceeded"

**Проблема:** Превышен лимит запросов SerpAPI

**Решение:**
1. Проверьте Dashboard SerpAPI
2. Увеличьте план или уменьшите частоту запусков
3. Измените Cron на `0 9 * * 1,4` (2 раза в неделю)

### Ошибка: "OpenAI rate limit"

**Проблема:** Слишком много запросов к GPT-4

**Решение:**
1. Добавьте ноду **"Wait"** перед GPT-4 (5-10 секунд)
2. Уменьшите количество товаров в фильтре
3. Используйте `gpt-4o-mini` вместо `gpt-4o` (дешевле)

### Workflow не запускается автоматически

**Проблема:** Schedule Trigger не активен

**Решение:**
1. Убедитесь, что workflow **Active** (зеленый переключатель)
2. Проверьте, что n8n запущен постоянно
3. Проверьте логи: Settings → Executions

### Нет результатов в Google Sheets

**Проблема:** Нет товаров, прошедших фильтр

**Решение:**
1. Снизьте критерии фильтрации (например, min_rating: 4.0)
2. Увеличьте диапазон цен (max_price: 100)
3. Измените категорию на более популярную
