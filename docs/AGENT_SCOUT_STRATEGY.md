## 🎯 Философия качественного продакт-ресерча

> **Главный принцип:** Лучше потратить 2 недели на поиск идеального товара, чем 2 месяца продавать провальный.

Профессиональный продакт-ресерч — это **не про скорость, а про точность**. Каждый найденный товар должен пройти минимум 7 слоев валидации.

## 🔬 7-слойная система валидации товара

### Слой 1: Первичный скрининг (автоматизируется)

**Критерии:**
- ✅ Рейтинг: ≥4.5 звезд
- ✅ Отзывы: >100 (для новых товаров) или >500 (для старых)
- ✅ Возраст товара: <6 месяцев (ищем ранние тренды)
- ✅ Цена: $15-$60 (sweet spot для импульсных покупок)
- ✅ Маржа: минимум 3x (продаем за $60 то, что стоит $20)

**Инструменты:** Sell The Trend NEXUS, AliExpress API

### Слой 2: Анализ тренда (TikTok)

**Что проверяем:**
- 📈 Количество упоминаний товара за последние 30 дней
- 📺 Топ-3 вирусных видео (>100k просмотров)
- 💰 Есть ли активные TikTok Shop продавцы?
- 🎯 Engagement rate (лайки/просмотры >5%)

**Инструменты:** Kalodata, FastMoss, TikTok Creative Center

### Слой 3: Конкурентный анализ

**Что смотрим:**
- 🏪 Сколько Shopify магазинов продают товар? (идеал: 5-20)
- 💵 Какие цены у топ-3 конкурентов?
- 📊 Есть ли данные о их revenue? (Ali Insider)
- 🎨 Какие креативы используют?

**Red Flags:**
- ⛔ >50 магазинов = перенасыщение
- ⛔ <3 магазинов = может не взлететь

**Инструменты:** Ali Insider, Dropship Spy

### Слой 4: Google Trends валидация

**Проверяем:**
- 📊 Растущий тренд последние 3-6 месяцев
- 🌍 Популярность в целевых странах (USA, UK, CA, AU)
- ❄️ Сезонность (избегаем товаров типа "новогодние украшения")
- 🔄 Долгосрочный потенциал (не одноразовый хайп)

**Инструмент:** Google Trends

### Слой 5: Supplier проверка

**Критерии:**
- ✅ Verified Supplier на Alibaba/1688
- ✅ MOQ (минимальный заказ) ≤50 шт
- ✅ Опыт работы: >2 лет
- ✅ Время производства: <15 дней
- ✅ Shipping: возможность ePacket/CJ Packet

**Инструменты:** Alibaba, CJDropshipping, AutoDS

### Слой 6: Юнит-экономика

**Расчет прибыльности:**

```
Себестоимость товара:     $12
Доставка (ePacket):       $3
Упаковка:                 $1
━━━━━━━━━━━━━━━━━━━━━━━━━━━
TOTAL COST:               $16

Цена продажи:             $49.99
Комиссия Shopify (2%):    $1.00
Payment processor (3%):   $1.50
Реклама (CPA):            $15
━━━━━━━━━━━━━━━━━━━━━━━━━━━
PROFIT:                   $16.49

Margin: 33% ✅
ROI: 103% ✅
```

**Минимальные требования:**
- Profit margin: >25%
- ROI: >80%

### Слой 7: Креативный потенциал

**Вопросы:**
- 🎥 Можно ли снять яркое демо-видео?
- 😮 Есть ли "WOW-эффект" при использовании?
- 📖 Решает ли реальную проблему?
- 💬 Легко ли объяснить пользу за 5 секунд?

**Примеры товаров с высоким креативным потенциалом:**
- ✅ До/после трансформация
- ✅ Решает боль (back pain, организация, экономия времени)
- ✅ Визуально эффектный процесс

## 🤖 Стратегия автоматизации Агента-Скаута

### Режим 1: Глубокий ресерч (1-2 товара в неделю)

**Цель:** Найти "бриллианты" с потенциалом $50k+/месяц

**Процесс:**
1. **День 1-2:** Сканирование 500+ товаров (Слой 1)
2. **День 3-4:** TikTok анализ топ-20 (Слой 2)
3. **День 5:** Конкурентный анализ топ-10 (Слой 3)
4. **День 6:** Google Trends + supplier check топ-5 (Слои 4-5)
5. **День 7:** Финальная валидация топ-2 (Слои 6-7)

**Результат:** 1-2 супер-валидированных товара

### Режим 2: Быстрый скрининг (5-10 товаров в неделю)

**Цель:** Быстро тестировать гипотезы

**Процесс:**
1. **Авто-сканирование** через API (Слои 1-2)
2. **Ручная проверка** топ-10 (Слой 3-4)
3. **Быстрый supplier check** (Слой 5)

## 🛠️ Технический стек для n8n

### Необходимые ноды:

1. **HTTP Request** - для API calls (Sell The Trend, Kalodata, etc.)
2. **Webhook** - триггер для запуска workflow
3. **Code** (JavaScript) - обработка данных, расчет метрик
4. **Google Sheets** - сохранение результатов
5. **Filter** - фильтрация по критериям
6. **Set** - трансформация данных
7. **Merge** - объединение данных из разных источников
8. **IF/Switch** - логика валидации
9. **Schedule Trigger** - автоматический запуск (например, каждый день)
10. **Email** / **Telegram** - уведомления о находках

### API которые будем использовать:

- ✅ **SerpApi** - для парсинга AliExpress, Amazon
- ✅ **Keepa API** - Amazon цены и история
- ✅ **Google Trends API** - трендовые данные
- ✅ **TikTok Creative Center** - через unofficial API/scraping
- ✅ **Sell The Trend API** - если доступен
- ✅ **OpenAI GPT-4** - анализ и генерация инсайтов

## ⚠️ Важные предупреждения

1. **НЕ гонитесь за количеством** - лучше 1 валидированный товар, чем 10 невалидированных
2. **Всегда заказывайте sample** - не продавайте то, что не держали в руках
3. **Следите за seasonality** - избегайте сезонных товаров без опыта
4. **Проверяйте авторские права** - особенно на брендовые товары
5. **Не игнорируйте юнит-экономику** - без прибыли бизнес не выживет
