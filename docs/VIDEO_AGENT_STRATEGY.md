## 📊 ИССЛЕДОВАНИЕ ЭФФЕКТИВНОСТИ 2026

### Статистика по форматам видео:

**UGC-Style Video (Победитель):**
- ✅ **3× выше CTR** чем брендированные ads
- ✅ **+161% конверсии** vs традиционный influencer контент
- ✅ **93% маркетологов** подтверждают доверие к UGC
- ✅ **+22% эффективнее** brand-created контента
- ✅ **50% engagement rate** для видео <1 минуты

**AI Avatars (НЕ для дропшиппинга):**
- ❌ **Lifestyle/Fashion: -30% engagement**
- ❌ **Fitness: -45% engagement**
- ❌ **Food: -35% engagement**
- ✅ Tech/Gaming: +15-25% engagement (не наш случай)

**Анимация vs Статика:**
- ✅ Видео: **+48% engagement** vs статичные изображения
- ❌ Статика: **-50% лайков, -38% шеров**
- ✅ AI-анимированный контент: **+40% completion rate**

**Алгоритм TikTok 2026:**
- Нужно **75%+ completion rate** для алгоритмического boost
- Удержание зрителя past **40-60% completion** = критично
- Порог engagement изменился с 3-5 сек на **15-20 сек** (декабрь 2024)
- Высокое разрешение + вертикальное видео + четкий звук = приоритет

**Бюджет и объем:**
- Нужно **10-20 вариаций** для нахождения winners
- Минимум **$20/день** на ad set
- AI UGC генераторы: **$39-99/мес** vs **$3000+** за видео у агентств

## 🎯 ВЫБРАННАЯ СТРАТЕГИЯ

### Формат: **UGC-Style Animated Video**

**Почему:**
1. Максимальная конверсия (161% vs традиционные методы)
2. Выглядит как контент от друга, не как реклама
3. Алгоритм TikTok 2026 приоритизирует видео над статикой
4. Можно масштабировать через AI tools

**Подход:** Создавать **2 версии** каждого креатива для A/B тестирования:
- **Версия A (Fast):** Статичные UGC фото + transitions + voiceover
- **Версия B (Premium):** Анимированное видео (Runway Gen-3) + advanced effects

## 🎬 ЧТО БУДЕТ ДЕЛАТЬ КАЖДЫЙ WORKFLOW

### Pipeline автоматизации:

```
1. INPUT (от пользователя):
   - Название продукта
   - Описание
   - Целевая аудитория
   - Фото продукта (опционально)

2. GPT-4o АНАЛИЗ (5-10 сек):
   - Анализ продукта
   - Определение pain points целевой аудитории
   - Создание viral стратегии (doomscroll hooks, ADHD-friendly, etc.)
   - Генерация TikTok скрипта (20-30 сек)

3. MIDJOURNEY ГЕНЕРАЦИЯ (60-90 сек):
   - Создание 4-8 UGC-стиль промптов
   - Генерация изображений через Discord API
   - Download результатов

4. ELEVENLABS ОЗВУЧКА (10-20 сек):
   - Конвертация скрипта в AI voice
   - Естественные интонации, filler words
   - Download MP3

5A. SHOTSTACK МОНТАЖ - Budget/Standard (30-60 сек):
   - Сборка изображений в timeline
   - Transitions между кадрами
   - Наложение voiceover
   - Музыка из библиотеки
   - Text overlays (captions)
   - Export MP4

5B. RUNWAY GEN-3 - Standard/Premium (120-180 сек):
   - Image-to-video анимация ключевых кадров
   - 5 секунд анимации на кадр
   - Download результатов

6. FINAL ASSEMBLY - Premium (30-60 сек):
   - Shotstack собирает:
     - Анимированные кадры от Runway
     - Статичные кадры от Midjourney
     - B-roll от Artgrid
     - Voiceover
     - Музыка
     - Captions, transitions, effects
   - Export финального MP4

7. OUTPUT:
   - Готовое MP4 видео (9:16, 1080×1920)
   - TikTok-ready (20-45 сек)
   - Скрипт + таймкоды
   - Midjourney промпты (для повторов)
   - Performance прогнозы (viral score, CTR estimate)
```

### Обязательные (для всех версий):
1. **OpenAI API** (уже есть)
   - GPT-4o для анализа и скриптов
   - ~$0.10-0.30 за видео

   - Генерация UGC-стиль изображений
   - Доступ через Discord API или unofficial API

   - AI озвучка
   - Starter/Creator/Pro в зависимости от версии

4. **Shotstack** ($0-99/мес)
   - Видеомонтаж API
   - Free (10 видео) / Standard / Pro

### Опциональные (Standard/Premium):
5. **Runway Gen-3** (~$12/видео)
   - Image-to-video анимация
   - Pay-per-use модель

6. **Epidemic Sound** ($15/мес)
   - Copyright-safe музыка
   - Unlimited downloads

7. **Artgrid** ($42/мес) - только Premium
   - Stock footage для B-roll
   - Cinematic качество

### Альтернативы бесплатным ресурсам:
- **Музыка:** TikTok Commercial Music Library (бесплатно)
