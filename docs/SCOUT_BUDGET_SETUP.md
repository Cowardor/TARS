## 📋 Пошаговая настройка (15 минут)

### Шаг 1: Получить OpenAI API ключ

1. Перейдите на: https://platform.openai.com/
2. Зарегистрируйтесь или войдите
3. **Settings** → **API Keys** → **Create new secret key**
4. Скопируйте ключ (начинается с `sk-...`)

**Важно:**
- Добавьте $5-10 на счет: **Billing** → **Add payment method**
- Проверьте usage limits: **Settings** → **Limits**

---

### Шаг 2: Получить RapidAPI ключ (бесплатно)

1. Перейдите на: https://rapidapi.com/apidojo/api/aliexpress-datahub
2. Нажмите **Sign Up** (можно через Google)
3. На странице API нажмите **Subscribe to Test**
4. Выберите **Basic Plan** → **$0/месяц** (100 запросов/день)
5. Скопируйте **X-RapidAPI-Key** справа

**Бесплатный лимит:**
- 100 запросов/день = достаточно для анализа 100 товаров/день
- Без ограничений по времени

---

### Шаг 3: Создать Google таблицу

1. Перейдите на: https://sheets.google.com/
2. Создайте новую таблицу
3. Назовите её: **TARS - Scout Results**
4. Скопируйте **ID таблицы** из URL:
   ```
   https://docs.google.com/spreadsheets/d/ВАШ_ID_ЗДЕСЬ/edit
   ```

---

### Шаг 4: Создать Telegram бота (опционально)

1. Откройте Telegram
2. Найдите **@BotFather**
3. Напишите `/newbot`
4. Следуйте инструкциям (имя, username)
5. Скопируйте **токен** (например: `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`)
6. Узнайте свой **Chat ID**: напишите **@userinfobot** и скопируйте ID

---

### Шаг 5: Импортировать workflow в n8n

1. Откройте n8n: http://localhost:5678
2. Нажмите **Import from File** (Ctrl+O)
3. Выберите файл: `TARS/workflows/scout-agent-v1-budget.json`
4. Workflow появится в редакторе

---

### Шаг 6: Настроить credentials в n8n

#### OpenAI Credentials

1. Кликните на ноду **"GPT-4o-mini: Анализ"**
2. **Credentials** → **Create New**
3. Выберите **OpenAI**
4. Вставьте ваш **API Key** (начинается с `sk-...`)
5. **Save**

#### RapidAPI Credentials

1. Кликните на ноду **"RapidAPI: Поиск на AliExpress"**
2. **Authentication** → **Generic Credential Type** → **Header Auth**
3. **Create New Credential**
4. **Credential Name:** `RapidAPI`
5. **Header Name:** `X-RapidAPI-Key`
6. **Header Value:** Ваш RapidAPI ключ
7. **Save**

Добавьте второй header:
1. В той же ноде добавьте в **Header Parameters**:
   - **Name:** `X-RapidAPI-Host`
   - **Value:** `aliexpress-datahub.p.rapidapi.com`

#### Google Sheets Credentials

1. Кликните на ноду **"Сохранить в Google Sheets"**
2. **Credentials** → **Create New**
3. Выберите **Google Sheets OAuth2**
4. Следуйте инструкциям OAuth (авторизация Google)
5. Вернитесь в n8n
6. **Document ID:** Вставьте ID вашей Google таблицы (из Шага 3)
7. **Save**

#### Telegram Credentials (опционально)

1. Кликните на ноду **"Уведомление в Telegram"**
2. **Credentials** → **Create New**
3. **Access Token:** Токен от @BotFather
4. **Chat ID:** Ваш Chat ID от @userinfobot
5. **Save**

---

### Шаг 7: Настроить параметры поиска

В ноде **"Установить параметры поиска"** измените:

```javascript
search_query: "smart home gadgets"  // Замените на вашу нишу
min_rating: 4.5      // Минимальный рейтинг
min_reviews: 100     // Минимум отзывов
min_price: 15        // Минимальная цена ($)
max_price: 60        // Максимальная цена ($)
```

**Популярные ниши для поиска:**
- `"fitness accessories"`
- `"pet grooming tools"`
- `"kitchen gadgets"`
- `"car accessories"`
- `"phone accessories"`

## 🔧 Troubleshooting

### Ошибка: "RapidAPI quota exceeded"

**Проблема:** Превышен лимит 100 запросов/день

**Решение:**
1. Измените расписание на реже (например, 2 раза в неделю)
2. Или апгрейдьте на Pro план ($10/месяц для 10,000 запросов)

### Ошибка: "OpenAI rate limit"

**Проблема:** Слишком много запросов к GPT-4o-mini

**Решение:**
1. Добавьте ноду **"Wait"** (5 секунд) перед GPT-4o-mini
2. Уменьшите количество товаров в фильтре (например, топ-20 вместо всех)

### Workflow не запускается автоматически

**Проблема:** Schedule Trigger не активен

**Решение:**
1. Убедитесь, что workflow **Active** (зеленый переключатель)
2. Проверьте, что n8n запущен постоянно
3. Проверьте логи: **Settings** → **Executions**

### Нет результатов в Google Sheets

**Проблема:** Нет товаров, прошедших фильтр

**Решение:**
1. Снизьте критерии (min_rating: 4.0, min_reviews: 50)
2. Измените search_query на более популярную нишу
3. Увеличьте max_price до $100

## 💡 Советы по оптимизации

### Экономия OpenAI токенов

1. **Батчинг:** Анализируйте по 5-10 товаров за раз (вместо по одному)
2. **Кэширование:** Не анализируйте один товар дважды
3. **Фильтрация:** Отправляйте в GPT только топ-20 товаров (после первичного фильтра)

**Ожидаемый расход:**
- 100 товаров/день × 500 токенов = 50,000 токенов/день
- 50,000 × 30 дней = 1.5M токенов/месяц
- Стоимость: 1.5M × $0.15 = **$0.225/месяц** (меньше доллара!)

### Увеличить качество находок

1. **Измените search_query** на более специфичную нишу
2. **Увеличьте min_reviews** до 500 для более валидированных товаров
3. **Добавьте ручную проверку** для STRONG BUY товаров
