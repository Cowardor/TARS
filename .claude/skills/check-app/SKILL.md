## Шаг 3 — Проанализируй результаты

Проверь по дизайн-системе (`docs/DESIGN_SYSTEM.md`):
- ✅ JS ошибки
- ✅ Хардкод цветов (должно быть 0)
- ✅ ViewBox иконок (должен быть 0 0 20 20)
- ✅ Шрифт (должен быть Switzer)
- ✅ Визуально: отступы, цвета, читабельность

Дай чёткий отчёт:
1. Найденные баги с указанием файла и строки
2. Визуальные проблемы со скриншотов
3. Список исправлений с приоритетами
